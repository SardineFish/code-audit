# Code Audit

A simple tool to provide code audit for C/C++.

The programing task for *Software Security*.

## Building Requirement
- CMake
- GCC with `make`
- Node v11.11.0+
- Python 2.7+

## Resolve Pakcages
```shell
$ npm install
```

## Build
```shell
$ npm run build
```

## Test
```shell
$ npm test
```

## Run

### Run main program
```shell
$ npm start
```

### Start distributed analyser node
```shell
$ ./distributed/bin/dist_node
```
